# コンテンツの投稿と閲覧

## 概要

このアプリケーションは、コンテンツの投稿と閲覧の実装例です。

## 機能

- テキスト、画像、動画を投稿することができます。
- 投稿したコンテンツを閲覧することができます。

[StackBlitz](http://stackblitz.com/) でプレビューを確認することができます。
